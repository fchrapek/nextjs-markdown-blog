Event Loop w JavaScript jest to mechanizm odpowiadający za zarządzanie kolejnością i wykonywaniem zadań asynchronicznych. Event Loop non-stop sprawdza, czy w kolejce zadań nie ma jakichś akcji do wykonania i jeśli tak, to je wykonuje. W ten sposób JavaScript jest w stanie wykonywać wiele zadań jednocześnie, bez blokowania interfejsu użytkownika. Wszystko to jest możliwe dzięki asynchronicznej natury języka, a Event Loop jest sercem tego procesu.
